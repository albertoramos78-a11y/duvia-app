# Duvia

Application de coparentalité — "Two homes. One family."

## Lancer en local

```
npm install
npm run dev
```

## Déploiement

Ce projet est conçu pour être déployé sur **Vercel** (déploiement automatique à chaque envoi sur GitHub).

Configuration Vercel :
- Build Command : `npm run build`
- Output Directory : `dist`
